تقرير تصاريح ومخالفات الحفر
================================
محتويات الحزمة:
  • index.html    : صفحة البداية المخصصة للنشر على Vercel وGitHub Pages
  • التقرير.html  : تقرير مرئي (افتحه في المتصفح)
  • البيانات.json : البيانات الكاملة بصيغة JSON
  • البيانات.csv  : البيانات بصيغة CSV (متوافق مع Excel)

عدد السجلات: 20

طريقة النشر:
  • ارفع جميع الملفات إلى جذر المستودع.
  • يجب إبقاء index.html في جذر المستودع حتى يعمل رابط الموقع الرئيسي دون خطأ 404.
